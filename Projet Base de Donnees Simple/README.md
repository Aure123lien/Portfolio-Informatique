# Projet Base de Donnees Simple

Ce projet présente une base de données simple créée grace au language SQL. Il s'agit d'une base de données pour gérer une bibliothèque avec des tables pour les livres et les auteurs célèbre

# Objectif

Démontrer mes compétences en conception et manipulation de bases de données, en créant un schéma et une ogarnisastion simple et en insérant des données de manière structurée

# Fichiers

- `schema.sql` : Script SQL pour créer les tables de la base de données
- `insert_data.sql` : Script SQL pour insérer des données d'exemple

# Comment recréer la base de données

1. Installer SQLite3 si vous ne l'avez pas encore
2. Exécuter `sqlite3 ma_base.db < schema.sql`
3. Puis  `sqlite3 ma_base.db < insert_data.sql`
